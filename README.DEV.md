# Useful commands

```shell
$ docker build -t aliengreenllc/mongo-restore-s3 .
$ docker run -t aliengreenllc/mongo-restore-s3
$ docker push aliengreenllc/mongo-restore-s3:latest

$ docker compose up -d --force-recreate --no-deps name
$ docker exec -it name sh

$ docker compose logs -f name
$ docker history name
```
